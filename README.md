# Pagination React App

A clean and interactive React application demonstrating efficient client-side pagination for large datasets. Features a responsive grid and intuitive navigation.

## Features

- **Client-Side Pagination**: Smoothly handle large datasets with dynamic page generation.
- **Dynamic Navigation**: "Previous", "Next", and numbered buttons for easy browsing.
- **Responsive UI**: Optimized for all device sizes, from desktop to mobile.
- **Efficient State Management**: Uses React hooks for predictable UI updates.

- **React**: Modern UI library for the frontend.
- **JavaScript (ES6+)**: Core logic and state handling.
- **CSS3**: Custom styling with responsive layouts.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aadhar41/react-pagination.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd pagination
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm start
   ```
