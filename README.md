# pagination

A simple React.js application demonstrating client-side pagination.

## Features

- Dynamic page navigation.
- Responsive UI components.
- Efficient data handling.

## Screenshots

![Home Screen](public/screens/home-screen.png)

## Getting Started

### Installation

```bash
npm install
```

### Usage

```bash
npm start
```

### Build

```bash
npm run build
```

## Technologies Used

- **React**: Frontend library for building the user interface.
- **JavaScript (ES6+)**: Core logic and state management.
- **CSS**: Custom styling for responsive layouts.

## Project Structure

```text
src/
├── components/     # Reusable UI components (Pagination, DataList)
├── App.js          # Main application logic and state
└── index.js        # Application entry point
```
